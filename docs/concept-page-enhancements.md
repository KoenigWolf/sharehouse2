# コンセプトページ機能拡張計画

## 概要

コンセプトページ (`/concept`) の機能拡張計画。SEREAL風のミニマルデザインを維持しながら、ユーザー体験とコンバージョンを向上させる。

---

## フェーズ 1: 必須機能

### 1.1 FAQ セクション

**目的**: 入居検討者の疑問を事前に解消

**実装内容**:
- アコーディオン形式の Q&A
- 5-8 項目程度
- スムーズな開閉アニメーション

**技術要件**:
- `@radix-ui/react-accordion` または shadcn/ui Accordion
- i18n キー: `concept.faq.items[].question`, `concept.faq.items[].answer`

**配置**: Principles セクションの後

---

### 1.2 住民の声（テスティモニアル）

**目的**: 社会的証明による信頼性向上

**実装内容**:
- 3-5 名の住民コメント
- アバター + 名前 + 居住期間 + コメント
- カルーセルまたはグリッド表示

**技術要件**:
- 匿名化オプション（イニシャル表示）
- Framer Motion でフェードイン

**配置**: Features セクションの後

---

### 1.3 ダークモード対応

**目的**: システム設定との一貫性

**実装内容**:
- `bg-white` → `bg-background` に変更
- `text-foreground/XX` の透明度調整
- 装飾円のボーダー色調整

**技術要件**:
- Tailwind の `dark:` バリアント使用
- CSS 変数ベースの色定義

---

## フェーズ 2: UX 改善

### 2.1 セクションナビゲーション

**目的**: 長いページでの現在位置把握

**実装内容**:
- 右側に固定ドットナビゲーション
- アクティブセクションのハイライト
- クリックでスムーススクロール

**技術要件**:
- Intersection Observer API
- `scroll-behavior: smooth`

---

### 2.2 スクロール進捗インジケーター

**目的**: 読了率の可視化

**実装内容**:
- ページ上部に細いプログレスバー
- スクロールに連動して伸縮

**技術要件**:
- `scrollYProgress` を流用
- `position: fixed; top: 0`

---

### 2.3 入居までの流れ

**目的**: オンボーディングの明確化

**実装内容**:
- 4-5 ステップのタイムライン
- アイコン + タイトル + 説明
- ホバーで詳細表示

**配置**: CTA セクションの前

---

## フェーズ 3: コンバージョン強化

### 3.1 お問い合わせフォーム

**目的**: リード獲得

**実装内容**:
- モーダルまたは専用セクション
- 名前 + メール + メッセージ
- バリデーション + 送信確認

**技術要件**:
- React Hook Form + Zod
- Supabase Edge Function または Resend API

---

### 3.2 住民数カウンター

**目的**: コミュニティ規模の可視化

**実装内容**:
- アニメーション付き数字カウントアップ
- 現在の住民数 / 最大定員を表示
- スクロールトリガーで起動

**技術要件**:
- `useInView` + `animate` (Framer Motion)
- Supabase からリアルタイム取得

---

### 3.3 イベントカレンダープレビュー

**目的**: コミュニティ活動のアピール

**実装内容**:
- 直近 3 件のイベントカード
- 日付 + タイトル + 参加者数
- 「すべて見る」リンク

**配置**: Features セクション内または後

---

## 実装優先順位

```text
Phase 1 (必須)
├── 1.3 ダークモード対応
├── 1.1 FAQ セクション
└── 1.2 住民の声

Phase 2 (UX改善)
├── 2.2 スクロール進捗インジケーター
├── 2.1 セクションナビゲーション
└── 2.3 入居までの流れ

Phase 3 (コンバージョン)
├── 3.2 住民数カウンター
├── 3.3 イベントプレビュー
└── 3.1 お問い合わせフォーム
```

---

## 技術スタック

| カテゴリ | 技術 |
|---------|------|
| アニメーション | Framer Motion |
| フォーム | React Hook Form + Zod |
| UI コンポーネント | shadcn/ui |
| i18n | 既存の useI18n フック |
| スタイリング | Tailwind CSS |

---

## 注意事項

- SEREAL 風のミニマルデザインを維持する
- アニメーション duration は 0.2-0.4s、easing は `[0.25, 0.46, 0.45, 0.94]`
- 全てのテキストは i18n 経由
- モバイルファースト設計
